## 一、机房日常巡检规范

### 1.1 每日巡检项目

**基础环境检查**
- 机房温度：正常范围 22±2°C
- 机房湿度：正常范围 40%-60% RH
- 空调运行状态：各机组出风温度、压力
- UPS 输入输出电压、负载率
- 消防系统烟感、温感状态

> **重要提示**：发现温湿度异常应第一时间排查空调故障，避免设备因高温宕机。

### 1.2 每周巡检项目

1. 电池组单体电压测量
2. 柴油发电机空载试机（30分钟）
3. 列头柜各支路电流检查
4. 机柜进风口滤网清洁度检查
5. 消防气瓶压力检查

### 1.3 月度巡检项目

- [x] UPS 系统深度放电测试
- [x] 空调冷凝器翅片清洗
- [x] 接地电阻测试
- [ ] 柴油发电机带载测试
- [ ] 消防系统联动测试

## 二、网络设备配置模板

### 2.1 核心交换机基础配置

```shell
# 进入系统视图
system-view
sysname Core-SW-01

# 配置管理VLAN
vlan 10
name Management
description Management_Network

# 配置VLANIF接口
interface Vlanif10
ip address 192.168.10.1 255.255.255.0

# 配置端口聚合
interface Eth-Trunk1
port link-type trunk
port trunk allow-pass vlan all
```

## 四、应急预案摘要

### 4.1 市电中断应急流程

1. **0-5分钟**：确认UPS供电正常，检查柴油发电机自启动状态
2. **5-10分钟**：发电机启动成功后，手动切换至发电机供电
3. **10-30分钟**：每5分钟检查一次油机运行参数
4. **30分钟后**：如市电仍未恢复，通知运维主管到场
5. **2小时后**：启动油料补充预案

### 4.2 空调故障应急处理

> **单台空调故障**：
> - 立即开启备用空调
> - 调整相邻空调出风角度
> - 密切关注机柜进风温度
> - 高温机柜可临时开启机柜门散热

> **多台空调同时故障**：
> - 立即通知空调厂商紧急到场
> - 非核心业务服务器临时关机
> - 移动工业冷风机进场支援
> - 必要时启动用户通知机制
